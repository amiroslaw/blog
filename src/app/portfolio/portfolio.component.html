Languide

Web application for learning foreign languages.

The web application for learning foreign languages. The application easily compares the texts from two languages sentence by sentence. This note could have tags and can be saved in a notebook. You can create a dictionary from select words.
Notice: This app is in developer stage.

Features:

  • Authentication a user
  • Comparing two texts
  • Creating notebook with user articles
  • Adding words to a dictionary
  • Managing user articles and notebooks
  • Saving to database

This app won't have translation option.

Used technology

Spring Boot, Spring MVC, Spring Security, Java 8, Hibernate, H2 database, JUnit, Mockito, Project Lombok, thymeleaf, Bootstrap 4, wordnik API

e-Wallet

Electronic wallet created in JavaFX technology

The electronic wallet was created using javaFX technology. Use it to manage your personal accounts. The application implements a Double-entry bookkeeping system. The program is cross-platform software, built in Java 8

Features:

  • creating a wallet
  • creating a user profile
  • logging in to a user profile
  • managing wallets
  • saving in to a database
  • creating reports

Used technology

Java 8, JavaFX, Scene Builder, SQLite, JDBC
The application was written in MVC pattern.
